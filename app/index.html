Simple AngularJS Template Application

This angularjs template provides a way to quickly get started with angularjs, leveraging the Gulpjs streaming build system. The application code itself is meant to support small and large projects by following Google's Best Practice Recommendations for Angular App Structure

AngularJS

Layout and Styling, Routing, and additional libraries

This sample utilizes bootstrap 3, providing just a simple sample layout. LESS is used for managing the CSS, with a main.less that does imports to control order, and separate .less files for mixins and variables, to support isolating customization. Also included are simple examples of other library use, like ui-router, restangular, and charting directives from angularjs-nvd3-directives.

Routing examples

Views are handled here by ui-router. Here are a couple of nested views, as well as a view directly accessed by menubar. UI-router provides a state machine, nested and parallel views. Example views are a simple to-do-form, an AngularUI-bootstrap component (accordian), and an nvd3.js chart (angularjs-nvd3-directives).
